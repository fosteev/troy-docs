# Classes — Классы персонажей

## Доступные классы

На старте: **Warrior** и **Mage**. В будущем планируется **Rogue**.

Каждый класс имеет свою **ресурсную систему**:
- **Warrior** — Rage (Ярость): накапливается в бою при ударах и получении урона
- **Mage** — Mana (Мана): постоянный ресурс, тратится на скиллы, регенерирует со временем

Скорость генерации/регенерации ресурса зависит от атрибута **Spirit**.

---

## Warrior (Воин)

**Роль**: ближний бой, высокая выживаемость, стабильный физический урон.

**Основные атрибуты**: Strength, Stamina.

**Ресурс**: Rage (0–100). Начинает бой с 0. Накапливается при автоатаке и получении урона. Тратится на скиллы. Вне боя сбрасывается в 0.

**Автоатака**: удар оружием (физический урон).

```
auto_damage = base_weapon_dmg + STR * 0.8
```

**Генерация ярости**:
```
При автоатаке:        rage += 5 * (1 + SPI * 0.02)
При получении урона:  rage += damage_taken * 0.15 * (1 + SPI * 0.02)
```

**Описание**: классический боец ближнего боя. Много HP и брони, стабильный урон. Не зависит от маны — чем дольше бой и чем больше урона получает, тем больше скиллов может использовать. Spirit ускоряет накопление ярости, позволяя чаще использовать способности.

## Mage (Маг)

**Роль**: высокий магический урон, большой запас маны, низкая выживаемость.

**Основные атрибуты**: Intelligence, Agility.

**Ресурс**: Mana. Максимум скейлится от Intelligence. Тратится на скиллы, регенерирует в бою со временем. Spirit ускоряет регенерацию. Каждый бой начинается с полной маны.

**Автоатака**: магическая стрела (магический урон).

```
auto_damage = base_spell_dmg + INT * 0.5
```

**Регенерация маны в бою**:
```
mana_regen = max_mana * 0.01 * (1 + SPI * 0.02) per second
```

**Описание**: кастер с огромным уроном, но хрупкий. Начинает бой с полной маной и может сразу использовать мощные скиллы. Должен грамотно расходовать ману — если закончится, останется только слабая автоатака. Spirit позволяет поддерживать ману в длительных боях.

## Сравнение ресурсных систем

| | Warrior (Rage) | Mage (Mana) |
|---|---|---|
| Старт боя | 0 — нужно накопить | Полная — можно сразу кастовать |
| Генерация | Автоатаки + получение урона | Регенерация со временем |
| Максимум | 100 (фиксированный) | Скейлится от INT |
| Влияние Spirit | +скорость генерации | +скорость регенерации |
| Вне боя | Не существует | Не существует (каждый бой с полной) |
| Стиль игры | Разгоняется со временем | Сильный старт, потом экономит |
